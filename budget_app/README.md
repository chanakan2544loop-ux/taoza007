# 🌸 Budgety — แอพจัดการงบประมาณส่วนตัว

## โครงสร้างไฟล์
```
budget_app/
├── app.py                   ← Flask backend (Python)
├── requirements.txt
├── budget.db                ← สร้างอัตโนมัติเมื่อรันครั้งแรก
├── templates/
│   ├── auth.html            ← หน้า Login / Register
│   └── dashboard.html       ← หน้าหลัก Dashboard
└── static/
    ├── css/
    │   └── style.css        ← สไตล์ Pastel ทั้งหมด
    └── js/
        ├── auth.js          ← Logic สำหรับ Login/Register
        └── dashboard.js     ← Logic Dashboard ทั้งหมด
```

## วิธีติดตั้งและรัน

### 1. ติดตั้ง Dependencies
```bash
pip install -r requirements.txt
```

### 2. รันแอพ
```bash
python app.py
```

### 3. เปิดเบราว์เซอร์
ไปที่ → http://localhost:5000

## ฟีเจอร์

| ฟีเจอร์ | รายละเอียด |
|---|---|
| สมัครสมาชิก | กำหนดชื่อ รหัสผ่าน และงบรายสัปดาห์ |
| เข้าสู่ระบบ | บันทึก session ด้วย Flask |
| บันทึกรายจ่าย | ชื่อรายการ จำนวนเงิน หมวดหมู่ |
| งบรายวัน | คำนวณจาก งบอาทิตย์ ÷ 7 |
| คงเหลือวันนี้ | แสดง progress bar สีแดงเมื่อเกินงบ |
| รายสัปดาห์ | แสดง bar chart รายจ่ายแต่ละวัน |
| รายเดือน | แสดง bar chart รายจ่ายทั้งเดือน |
| ลบรายการ | กดปุ่ม ✕ ได้เลย |
| ตั้งค่างบ | เปลี่ยนงบรายสัปดาห์ได้ตลอด |

---

## การคำนวณงบ

- **งบรายวัน** = งบรายสัปดาห์ ÷ 7
- **งบรายเดือน** = งบรายสัปดาห์ × 4.3
- **คงเหลือ** = งบวันนี้ − ยอดใช้จ่ายวันนี้
- เงินที่ใช้แต่ละวันไม่ยกยอดข้ามวัน (งบรีเซ็ตทุกวัน)
